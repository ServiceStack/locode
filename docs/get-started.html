Getting started with Locode #

ServiceStack Locode App is built into ServiceStack application and provides a user-friendly interface to manage data using AutoQuery services. It supports 4 major RDBMS providers including PostgreSQL, MS SQL, MySQL and SQLite. AutoQuery services can be generated directly from your database schema or use Plain Old C# Objects (POCOs) and infer CRUD behaviour as well as be extended and customized using C# attributes.
